# Projects

## Loan Default Dataset — анализ и предсказание дефолта по кредитам

Полный пайплайн работы с данными: от разведочного анализа (EDA) до обучения
и сравнения моделей классификации. Включает обработку пропусков и выбросов,
кодирование категориальных признаков, масштабирование, восстановление баланса
классов и корреляционный анализ. Цель — предсказать вероятность дефолта заёмщика.

> **Примечание:** ноутбук рассчитан в первую очередь на запуск в Google Colab /
> Kaggle, где большинство библиотек уже предустановлены. Версии пакетов в
> `pyproject.toml` приведены для локального запуска и со временем могут устареть.
